About Job Listing app

This Job Portal application can be used to search jobs on various domains like UX , Software Development , Hr , IT , Sales etc. A new user can first register him/herself easily and then login using the credentials. Then the user can search for various jobs on the basis of different categories and simultaneously add relevant jobs to his favourites list.

You can now login anytime and start applying for jobs by viewing yours favourites list. The job description is being fetched by the API Link.
